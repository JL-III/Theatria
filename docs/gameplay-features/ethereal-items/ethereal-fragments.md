# Ethereal Fragments

Fragments are obtained within [meteorite](../../events-challenges/meteorites.md) treasure barrels.

After collecting 16 Ethereal fragments you can trade them for an ethereal tool [here](https://discord.gg/GxD9msKtZD)!
