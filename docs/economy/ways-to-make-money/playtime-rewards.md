# Daily Rewards

### How DailyRewards Work

{% hint style="warning" %}
_Alts are not to be used with daily rewards_&#x20;
{% endhint %}

**Play and Earn:** Accumulate 1 hour of active playtime to earn 5 DailyRewardCrate keys.&#x20;

**AFK Detection:** Only active playtime counts; AFK time is excluded.&#x20;

**Daily Reset:** Progress resets daily at 0:00 UTC.&#x20;





**Check Your Progress:** Use the command `/daily-reward` to see how close you are to earning your keys!
