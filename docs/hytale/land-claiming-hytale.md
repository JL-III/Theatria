# Land Claiming (Hytale)

Land claiming in Hytale is very simple.

1. Stand in the chunk where you plan to claim (usually where you base is located.
2. Type `/sc claim`&#x20;
3. Use the map key to see where you've claimed. Default binding is `m`.
