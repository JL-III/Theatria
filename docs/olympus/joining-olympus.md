# Joining Olympus

### Joining Olympus

Olympus is whitelist-only. **We are not currently accepting applications.**

Players must be approved before joining. This protects the world, the community, and the long-term nature of the server.

After being accepted, install the official Olympus Mod Pack through Modrinth and connect using the server information provided in Discord.

### Required Modpack

To join Olympus, install the official modpack:

[Olympus Mod Pack on Modrinth](https://modrinth.com/modpack/olympus-theatria-mod-pack)

The modpack is required. A vanilla client cannot connect to Olympus.
