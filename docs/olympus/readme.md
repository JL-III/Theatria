# README

## Olympus

{% endhint %}

Olympus is the modded branch of **Theatria**.

It is a whitelist-only, long-term Minecraft survival server built around exploration, building, engineering, discovery, and player-driven projects. Olympus uses the official [Olympus Mod Pack](https://modrinth.com/modpack/olympus-theatria-mod-pack), which is required to connect.

Olympus is not a quest pack, not a seasonal reset server, not pay-to-win, and not built around command-based convenience.

It is a survival world where players create the story through what they build, automate, explore, trade, and preserve.

### What Olympus Is

Olympus expands Minecraft while keeping the core survival experience recognizable.

The server adds new dimensions, expanded terrain, Create-powered engineering, player shops, claims, voice chat, and custom Olympus content.

The goal is not to overwhelm players with every possible modded feature. The goal is to create a focused modded survival world with enough depth to support years of play.

### The Core Idea

Theatria began from a simple idea:

Player work should matter.

Olympus carries that idea into modded Minecraft. Players should be able to build towns, roads, bases, machines, farms, ships, airships, shops, and infrastructure without feeling like everything is temporary or meaningless.

The server is designed around permanence, restraint, and discovery.

### Who Olympus Is For

Olympus is for players who want:

* A long-term modded Minecraft world
* A whitelist-only survival community
* Building and infrastructure projects that matter
* Exploration without a fixed quest path
* Create engineering and automation
* Player-driven trade
* A world designed to last
* A world with history

Olympus may not be for players who want:

* Fast teleportation
* Server-run progression
* Heavy admin assistance
* Pay-to-win ranks
* Questbook-driven gameplay
* A short-term seasonal server
* A fully guided experience

### The Relationship Between Theatria and Olympus

Theatria is the main permanent survival home of the community.

Olympus is the modded branch.

Olympus exists for players who want a deeper modded survival experience while preserving the same underlying values: permanence, creativity, fairness, and player-driven history.

### Final Note

Olympus is not meant to be consumed quickly.

It is meant to be lived in.

Build something worth finding later.
